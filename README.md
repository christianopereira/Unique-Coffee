# Unique Coffee — Website Institucional

Website institucional premium para a [Unique Coffee](https://uniquecoffee.pt), cafeteria de especialidade em Caldas da Rainha, Portugal.

## Stack

- **Next.js 14** (App Router, TypeScript)
- **Tailwind CSS** (design system custom)
- **Framer Motion** (animações subtis)
- **Lucide React** (ícones)

## Desenvolvimento

```bash
# Instalar dependências
pnpm install

# Servidor local
pnpm dev

# Build de produção
pnpm build
```
